# 🃏 Blackjack Game (Python)  

A simple **console-based Blackjack game** built with Python.  

Play against the computer and try to beat the dealer without going over 21! ♠️♥️♣️♦️  

---

## 📌 About the Project  

<details>
<summary><strong>▶ What does this project do?</strong></summary>

This project is a terminal-based implementation of the classic **Blackjack (21)** card game.

The player competes against the computer (dealer):

- 🎴 Cards are dealt randomly  
- 🧮 Scores are calculated automatically  
- 🃏 Blackjack (21 with two cards) is handled  
- ♻️ Aces can count as **11 or 1**  
- 🏆 The winner is decided based on Blackjack rules  

This beginner-friendly project helps practice:

- 🧠 Functions  
- 🔁 `while` loops  
- 🔀 Conditional logic  
- 📋 Lists  
- 🎲 Randomization  
- 🐍 Game logic design  

</details>

---

## ✨ Features  

- 🎴 Random card dealing  
- 🃏 Blackjack detection  
- 🔄 Ace value adjustment (11 → 1)  
- 🤖 Computer dealer logic (hits until 17)  
- 🏆 Win / Lose / Draw outcomes  
- 🔁 Option to replay the game  
- 🎨 ASCII art logo
